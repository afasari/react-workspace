## üöÄ Quick Start

1. Clone the repository:

   ```bash
   git clone https://github.com/afasari/react-workspace.git
   cd calculator
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```
